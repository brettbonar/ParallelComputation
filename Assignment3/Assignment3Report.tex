\section{Implementation}
The master process with rank 0 splits the input array into equally sized arrays to be consumed by every process.

MPI Scatter is used to send the sublists to each process and an MPI Gather is used to collect the sorted sublists into the master process.

The master process then merges the sorted lists and outputs the final result.
